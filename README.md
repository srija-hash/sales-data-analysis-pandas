# Sales Data Analysis using Pandas

## Tools Used
Python, Pandas

## Description
Performed data cleaning and analysis on a sales dataset using Pandas.

## Work Done
- Handled missing values
- Filtered and sorted data
- Grouped data using groupby()
- Performed aggregations (sum, mean)

## Key Insights
- Identified top-selling products
- Found monthly sales trends
